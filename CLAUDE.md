# 프로젝트: 업무관리 풀스택 웹앱

## 기술 스택
- 서버: FastAPI + Python 3.11 + SQLite
- 프론트: Vanilla JS + Tailwind CDN
- 폴더: `backend/` 와 `frontend/` 분리

## 코딩 규칙
- 한국어 주석 사용
- 변수명: snake_case (Python), camelCase (JS)
- 모든 API는 `/api/` 경로 prefix 사용

## 금지사항
- jQuery 사용 금지
- CSS 직접 작성 금지 (Tailwind만 사용)
